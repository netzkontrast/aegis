## Notes on Methodology

This log is **research data**. The AEGIS project's core question is whether formal systems can serve creativity at novel-length scale. This log tracks empirical observations during the actual creative process.

**Be honest**:
- If ARCHON doesn't help, say so
- If tools feel like burden, document it
- If intuition beats structure, acknowledge it

**Be specific**:
- Don't write "it helped"—write "ncp_query helped by X"
- Don't write "it was frustrating"—write "consulting PROJECT_CODEX mid-scene broke flow because Y"

**Be consistent**:
- Write after each chapter while experience is fresh
- Use the template questions
- Track changes over time

**Purpose**:
This log will become the foundation for:
- ARCHON 2.0 (tools that actually help)
- Research paper on AI-assisted narrative
- Understanding of what AI collaboration looks like in practice
