# Integration Workflows

## Standard Integration Steps

1.  **Extract Content**: Use `tapestry` to get raw text.
2.  **Generate Plan**: Use `ship-learn-next` to create actionable steps.
3.  **Create Source Note**: Link raw content and plan.
4.  **Extract Concepts**: Create atomic Zettel notes.
5.  **Create MOC**: Link everything together.
6.  **Update Vault**: Add to index and cross-link.

## Advanced Integration

*   **PDF Processing**: Convert PDF to text before running tapestry.
*   **Video Processing**: Use transcripts for video content.
*   **Code Repos**: Analyze README and code structure for concept extraction.
