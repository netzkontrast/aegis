# AEGIS - Vercel Prototype

A web-based prototype showcasing the ARCHON narrative coherence framework and the Kohärenz Protokoll novel.

## Features

- **Home**: Project overview and navigation
- **NCP Query**: Interactive interface for querying the Narrative Context Protocol
- **Characters**: Explorer for Kael's 11 dissociative alters
- **World**: Kernwelten simulation layers and metaphysics
- **Manuscript**: Chapter viewer with navigation
- **About**: ARCHON framework documentation

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view in your browser.

## Deploy on Vercel

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com).

## Tech Stack

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI**: React 19
- **Deployment**: Vercel
