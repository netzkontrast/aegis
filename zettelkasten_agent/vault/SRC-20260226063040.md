# Cluster: ARCHON (Narrative Framework)

This cluster contains the core framework for Narrative Coherence Protocol (NCP). It defines the schemas, validation logic, and agent interfaces that ensure the story remains consistent with its own metaphysics.

## Core Directories

- **`ARCHON/ncp/`**: The heart of the system.
  - `schema.json`: The JSON Schema defining the structure of scenes, characters, and narrative beats.
  - `kohaerenz_protokoll.ncp.json`: The specific instance of the schema for this novel.
- **`ARCHON/tools/`**: Python scripts for interacting with the manuscript.
  - `ncp_validate.py`: Validates manuscript files against the NCP schema.
  - `ncp_query.py`: Allows querying the narrative database.
  - `ncp_assist.py`: Helper tool for authors.
- **`ARCHON/agents/`**: Configuration for AI agents operating within the framework.

## Key Files
- `ARCHON/INTERFACE_DESIGN.md`: Specification for how the system interacts with the user/author.
