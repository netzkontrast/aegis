# Cluster: Kohärenz Protokoll (Manuscript)

This cluster contains all creative assets for the novel *Kohärenz Protokoll*. It includes the actual prose, worldbuilding data, and detailed narrative plans.

## Core Directories

- **`kohaerenz_protokoll/manuscript/`**: The novel text.
  - `act_1/`: Chapters 1-13 (Fragmentation).
  - `act_2/`: Chapters 14-26 (Confrontation).
- **`kohaerenz_protokoll/narrative_design/`**: Structural planning documents.
  - `ACT_1_RECAP_AND_ACT_2_PLAN.md`: Strategic overview for the current act.
  - `chapters_7_13_outlines.md`: Specific outlines for late Act 1.
- **`kohaerenz_protokoll/world/`**: Encyclopedia of the setting.
  - `kernwelten/`: Details on KW1-KW4.
  - `characters/`: Character sheets for PKPs and NPCs.

## Key Files
- **`kohaerenz_protokoll/PROJECT_CODEX.md`**: The canonical "Bible" of the universe. Contains all metaphysical rules.
- `kohaerenz_protokoll/WRITING_PROTOCOL.md`: Guidelines for tone, style, and formatting.
- `docs/kohaerenz_protokoll_master_skill.md`: The Master Skill document encapsulating all rules for AI generation.
