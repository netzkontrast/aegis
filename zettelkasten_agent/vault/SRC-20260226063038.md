# Cluster: Web Application (Vercel Prototype)

This cluster contains the Next.js interactive prototype for the novel. It serves as the primary reader interface.

## Core Directories

- **`vercel-prototype/`**: The web application source code.
  - `src/app/`: Next.js pages and routes.
  - `public/`: Static assets.
  - `styles/`: Tailwind CSS configuration.

## Key Files
- `package.json`: Dependencies and scripts (`npm run dev`, `npm run build`).
- `README.md`: Instructions for setting up and running the app.
