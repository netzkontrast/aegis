# Cluster: Tools & Agents

This cluster contains the Python-based tools and AI agents that support the authoring and development process.

## Core Directories

- **`skill_seeker/`**: A powerful tool for converting documentation and repositories into AI Skills.
  - `cli/`: Command-line interface logic.
  - `docs/`: Extensive documentation for the Skill Seeker tool.
  - `configs/`: Scraper configurations for various frameworks (e.g., Python, React).
- **`zettelkasten_agent/`**: An MCP-based agent for managing knowledge.
  - `vault/`: The Zettelkasten note storage.
  - `schemas/`: Note types and definitions.
  - `zettelkasten_tools_mcp.py`: The MCP server for the agent.
- **`skills/`**: A library of reusable AI skills generated by Skill Seeker.
  - `skill-authoring/`: Skills for writing other skills.
  - `zettelkasten-tapestry/`: Skills for knowledge synthesis.
- **`.claude/`**: Configuration for Claude Code integration.

## Key Files
- `AGENTS.md`: The authoritative guide for all AI agents working in this repository.
- `skill_seeker/README.md`: How to use the Skill Seeker tool.
- `zettelkasten_agent/README.md`: How to run the Zettelkasten agent.
