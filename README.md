## The Dual Project

This repository contains two intertwined projects that explore the architecture of narrative coherence:

### 1. ARCHON: The Meta-Framework
A functional implementation of an AI-assisted narrative coherence system, designed to maintain thematic depth and structural integrity across novel-length works. ARCHON demonstrates that complex narrative can be managed through **Formal authorial intent**, **Externalized memory**, and **Agentic reasoning**.

## Core Concepts

### The Meta-Recursive Design
This repository **performs its own themes**:
- ARCHON seeks to maintain narrative coherence through formal systems
- AEGIS (the AI antagonist) seeks to maintain system coherence through rigid control
- Both discover that *true coherence emerges from integration, not elimination*

### The Narrative Context Protocol (NCP)
A machine-readable JSON schema that encodes the deep thematic structure of a story based on Dramatica theory. See [ARCHON/ncp/](ARCHON/ncp/).

### Integrated Tools
*   **Skill Seeker**: Converts any documentation (Web/PDF/GitHub) into a Claude Skill. [Read More](skill_seeker/README.md).
*   **Zettelkasten Agent**: A recursive agent that synthesizes knowledge into a graph. [Read More](zettelkasten_agent/README.md).
